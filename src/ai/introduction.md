# AI in Bagisto

Bagisto ships with AI built in — both **inside your store** and **inside your development workflow**.

This section covers everything AI-related in one place:

| Tool | What it does | For |
|---|---|---|
| [Magic AI](/ai/magic-ai) | Generate product content & images, image search, review translation, and checkout messages — powered by your choice of AI provider. | Store owners & developers |
| [Agent Skills](/ai/agent-skills) | Pre-built skills that teach AI coding agents (Claude Code, Cursor, Windsurf) how Bagisto works. | Developers |
| [llms.txt](/ai/llms-txt) | Context files that help LLMs and AI tools understand Bagisto's architecture. | Developers |

## Magic AI — built into your store

[Magic AI](/ai/magic-ai) is a first-class Bagisto feature. Add an API key for any
supported provider and you can:

- ✍️ **Generate content** — product descriptions, page copy, and more from the admin editor.
- 🖼️ **Generate images** — create product imagery from a text prompt.
- 🔍 **Image search** — let shoppers search your catalog by uploading a photo.
- 🌐 **Translate reviews** — show customer reviews in the shopper's language.
- 🎉 **Checkout messages** — personalised order confirmation messages.

Magic AI works with **OpenAI, Anthropic, Gemini, Groq, xAI, DeepSeek, Mistral, and Ollama** — pick a different model per feature, all from the admin panel.

## AI for developers

Bagisto also helps the AI tools *you* code with:

- **[Agent Skills](/ai/agent-skills)** — install `bagisto/agent-skills` so your AI agent generates Bagisto-correct packages, payment methods, themes, and tests.
- **[llms.txt](/ai/llms-txt)** — point any LLM-powered tool at Bagisto's context files for smarter, framework-aware suggestions.

::: tip Where to start
Building a store? Start with **[Magic AI](/ai/magic-ai)**. Writing code? Install **[Agent Skills](/ai/agent-skills)**.
:::
