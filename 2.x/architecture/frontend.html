# Vue and Tailwind

Vue.js (opens new window) and Tailwind CSS (opens new window) handles the complete UI of Bagisto.

Vite (opens new window) is a tool that offers a fast and efficient development environment for building modern web applications. It replaces Laravel Mix (opens new window), which was used for defining webpack (opens new window) and building steps in your Laravel (opens new window) application using various CSS and JavaScript pre-processors (opens new window).

Bagisto is using Vue.js (opens new window) and Vite (opens new window), vite compiles all of CSS and JavaScript assets that are placed into public directory.

In vite.config.js file, just define your Vue.js (opens new window) and path of build directory, Vite (opens new window) will be responsible to compile all the assets on defined location.

In tailwind.config.js file, just define your blade file path along with .js file directory, Tailwind CSS (opens new window) will be responsible to compile all the css defined on the location.

# Blade

Bagisto uses blade template engine that is included with Laravel. It does not restrict you from using plain PHP code in your templates. This documentation (opens new window) provides an explanation of directory structure and package configuration for implementations.
